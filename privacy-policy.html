Política de Privacidade

Última atualização: 23 de junho de 2026

1. Informações Coletadas

O aplicativo Agência Noir coleta os seguintes dados:

  • Dados de conta: e-mail e nome (fornecidos no cadastro ou via Google OAuth)
  • Dados de uso: registros de investigação, anotações, compromissos, histórico de chat, dados de estudo e demais conteúdos criados pelo usuário dentro do aplicativo
  • Dados do dispositivo: identificadores anônimos de sessão para funcionamento da autenticação

2. Como os dados são coletados

  • Fornecidos pelo usuário: formulários de cadastro, login, criação de conteúdo (anotações, casos, etc.)
  • Armazenamento local: dados salvos no navegador/dispositivo via localStorage e IndexedDB para funcionamento offline
  • Autenticação: via Supabase Auth (email/senha ou Google OAuth)

3. Finalidade do tratamento

  • Autenticar o usuário e permitir acesso à conta
  • Sincronizar dados entre dispositivos (cloud sync)
  • Garantir backup dos dados do usuário
  • Melhorar a experiência do aplicativo

4. Compartilhamento com terceiros

Não compartilhamos dados pessoais com terceiros. A infraestrutura utiliza:

  • Supabase (plataforma de backend open-source) — armazenamento de dados e autenticação
  • Google OAuth — login opcional via conta Google (apenas se o usuário escolher)

5. Armazenamento e segurança

Os dados são armazenados em servidores Supabase com criptografia em trânsito (TLS) e em repouso. O usuário também pode optar por manter dados apenas localmente (sem sincronização na nuvem).

6. Direitos do usuário (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a exclusão dos dados
  • Revogar o consentimento a qualquer momento
  • Exportar seus dados em formato estruturado

7. Exclusão de conta e dados

Para excluir sua conta e todos os dados associados, entre em contato pelo e-mail abaixo ou utilize a opção de exclusão dentro do aplicativo (se disponível). Os dados serão removidos em até 30 dias.

8. Dados offline

O aplicativo funciona offline para a maioria das funcionalidades. Dados salvos localmente permanecem no dispositivo até serem sincronizados com a nuvem ou removidos pelo usuário.

9. Permissões do dispositivo

O aplicativo solicita as seguintes permissões necessárias para seu funcionamento:

  • Internet — sincronização na nuvem
  • Câmera — captura de evidências
  • Microfone — gravação de notas de áudio
  • Notificações — lembretes de compromissos
  • Vibração — alertas
  • Wake Lock — manter tela ativa durante perseguições

10. Consentimento

Ao utilizar o aplicativo, você consente com esta política de privacidade. Você pode revogar o consentimento a qualquer momento excluindo sua conta e desinstalando o aplicativo.
