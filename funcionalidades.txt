1. SISTEMA CENTRAL
--------------------------------------------------------------------------------
- Login/Registro com autenticacao local (localStorage), log de debug
- reCAPTCHA-like bot detection (mousemove + time + checkbox) no login
- Modo Administrador (usuario: TURING / senha: ADM-SHADOW-X) — desbloqueia todas as abas
- Tema noir com tipografia Oswald/DM Mono, som de maquina de escrever
- Splash screen com fallback 5s
- Navegacao por abas com scroll horizontal infinito
- Persistencia total via localStorage
- Sistema de whiskey (moeda interna) — ganha 5/dia, gasta para desbloquear livros
- Activity Log (noirLogEvent) — registro cronologico de eventos no Dashboard
- Dashboard com cards de status (VPN, Geo, DND, Grayscale, Selfie, SMS, Audio, Timer, SoundMeter, Compass)
- Theme Switcher: 4 temas via CSS variables (Noir/red, Matrix/green, Amber/orange, Cyber/blue)
- PIN Lock overlay (4-6 digitos, hash simples, auto-lock no load)
- Biometric unlock (API 28+, androidx.biometric via reflection)

3. VIGILANCIA — PLUGINS NATIVOS (Android)
--------------------------------------------------------------------------------
3.1. SHUTDOWN TIMER
     - Agendamento de desligamento do sistema por comando (PluginMethod)
     - Cancelamento

3.2. ESCALA DE CINZA (Grayscale)
     - Aplica/remove filtro grayscale no dispositivo (Settings.System)
     - Alterna via toggle

3.3. NAO PERTURBE (DND)
     - Ativa/desativa NotificationManager.setInterruptionFilter
     - Status ON/OFF

3.4. INTRUDER SELFIE (Camera2)
     - Captura selfie com camera frontal ao errar senha
     - Salva em filesDir/selfies/ com .nomedia
     - Mirror na imagem capturada
     - Armado/desarmado via botao

3.5. GEO-FENCE (LocationManager)
     - addProximityAlert com raio de 50m a 5km
     - BroadcastReceiver para transicoes de entrada/saida
     - Log de alertas no HTML

3.6. SMS GUARDIAN (SmsManager)
     - Envio de SMS para numero de backup
     - Listener de SMS recebido com comandos: KILL, SELFIE, LOCATION
     - Emite eventos JS via notifyListeners

3.7. REMOTE KILL
     - Wipe total: deleta filesDir, cache, external files, cookies, WebStorage
     - Mata processo (Process.killProcess)
     - Confirmacao visual

3.8. AUDIO RECORDER (MediaRecorder)
     - Gravacao em AAC/MP4 no filesDir/recordings/
     - Listar, deletar gravacoes
     - Status ON/OFF

3.9. WI-FI SCANNER (WifiManager)
     - Scan de redes proximas (SSID, BSSID, level, frequency)
     - Fallback de 10s timeout
     - Exibicao com barras de sinal

3.10. FLASHLIGHT / SOS (CameraManager.setTorchMode)
      - Liga/desliga lanterna
      - Padrao SOS (3 curtos, 3 longos, 3 curtos — 200ms)
      - Strobe (100ms blink)
      - Cancela padrao

3.11. SOUND METER (AudioRecord)
      - Leitura PCM periodica (200ms)
      - Evento JS soundLevel com level% + dB
      - One-shot getSoundLevel

3.12. COMPASS (SensorManager)
      - Leitura de azimuth com acelerometro + magnetometro
      - Direcao cardeal (N/NE/E/SE/S/SW/W/NW)
      - Timeout de 3s

3.13. BIOMETRIC AUTH (BiometricPrompt API 28+)
      - Verifica disponibilidade via reflection (canAuthenticate)
      - Autenticacao com CancellationSignal + Executor
      - Fallback via PIN se indisponivel

4. ANTI-FORENSIC (AntiForensicPlugin.java)
--------------------------------------------------------------------------------
4.1. HADES — KeyStore Ephemeral
     - Geracao de chave AES-256 no AndroidKeyStore
     - Cifra/decifra com AES/GCM/NoPadding (IV + ciphertext base64)
     - Wipe total: destroi todas as chaves e dados do app

4.2. POLTERGEIST — Detonador Automatico
     - BroadcastReceivers: USB, SIM, fone, rede, energia
     - Sensor de acelerometro para deteccao de shake
     - Agendamento de wipe automatico com delay config (2/5/10s)
     - Eventos JS: poltergeistAlert, poltergeistWipeDone

4.3. COVERTRACKS — Limpeza de Rastros
     - Limpa clipboard, logcat, recents, cache do teclado
     - Remove downloads do DownloadManager

4.4. SENTINELA — Deteccao Forense
     - Detecta: root, USB debugging, modo desenvolvedor, emulador
     - Detecta: screen recording/mirroring, ADB network, USB conectado
     - Threat level (NONE/BAIXO/MEDIO/ALTO)
     - Monitoramento continuo (10s) com eventos JS sentinelaThreat

5. CRIPTO — MODULOS DE SEGURANCA
--------------------------------------------------------------------------------
5.1. BURNER NOTES (Android KeyStore + Web Crypto)
     - Tenta cifra nativa primeiro; fallback para Web Crypto API
     - AES-256/GCM, chave armazenada no AndroidKeyStore
     - Ciphertext + IV em base64

5.2. ESTEGANOGRAFIA LSB
     - Oculta mensagem em imagens PNG/BMP
     - Extrai mensagem oculta
     - 100% client-side (canvas API)

5.3. SECURE FILE SHREDDER
     - DoD 5220.22-M (3 passagens)
     - Gutmann (35 passagens)
     - Tenta nativo (SAF + FileChannel) primeiro; fallback web
     - Barra de progresso + log
